## Features

### Admin Dashboard

**Category Management**
- Add, edit, and delete categories and subcategories
- Manage category logos and sliders

**Product Management**
- Add, edit, and delete products
- Multiple product images per listing
- Set featured products
- Product descriptions, reviews & ratings
- Stock management and discount/offer settings
- Related product selection

**Order Management**
- View all orders
- Track awaiting payment and awaiting delivery orders

**Payment Settings**
- Add, edit, and delete payment methods
- Manage payment method icons
- PayPal and EFT (FNB) integration

**CMS / Website Settings**
- General and logo settings
- Home slider and banner management
- Social links and footer management
- Brand logo and category slider settings

---

### Customer Frontend
- Fully responsive design (mobile friendly)
- Product browsing, search, and filtering
- Product comparison
- Product ratings and reviews
- User registration and login
- Shopping cart and checkout
- Order tracking
- Social sharing
- SEO friendly URLs
- Newsletter subscription

## Installation

1. Clone the repository
2. Move the project folder to `/Applications/MAMP/htdocs/`
3. Create a MySQL database named `fashiony`
4. Import `fashiony_ogs.sql` via phpMyAdmin
5. Update `admin/admin/inc/config.php` with your DB credentials
6. Visit `http://localhost:8888/PHP-MySQL-ecommerce-website-master`
